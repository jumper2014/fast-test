## 项目简介

`fast-test` 是一个用来学习并实战自动化测试工具的项目。

## 开发环境

- **JDK 1.8 +**
- **Maven 3.5 +**
- **IntelliJ IDEA Community 2021.2.2 +** (*注意：务必使用 IDEA 开发，同时保证安装 `lombok` 插件*)
- **Mysql 5.7 +**
