## 项目简介

`fast-test` 是一个开箱即用的自动化测试工具项目，帮你快速构建自己的自动化测试框架，自动化测试工具，自动化测试平台。

## 开发环境

- **JDK 1.8 +**
- **Maven 3 +**
- **IntelliJ IDEA Community 2021.2.2 +** 
- **Visual Studio Code 1.63.2 +**
